Welcome to Pertino Python SDK
=============================

To use
######
  include file here

To Build
########
.. code-block::

  ant env
  ant init
  ant package

To run unit tests
#################
.. code-block::

  ant test

To run acceptance test
######################

  Create a file with your Pertino credentials (see acceptance/conftest.py for format/name)
.. code-block::

  ant acceptance
